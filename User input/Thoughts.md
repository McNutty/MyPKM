# Conventions

We should use DSRP terms wherever possible, so for example "nested cards" should be referred to as "parts", and so on.

# Notes

I love the automatic assignment of colors depending on nesting level. That is genious and I definitely want to keep it. One of the goals is for the user to spend as little time as possible "arranging" and "coloring", instead solely focused on "thinking". Every design choice should encourage "getting in the thinking zone".

Maybe allow for choosing a color scheme with matching colors? Instead of choosing individual card colors?

Also, I love the shape of the cards. A little rounded at the edges, very Plectica-like.
